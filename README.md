## Overview
 
This Android project is a base code for any MVVM project using Jetpack Compose applying Modularity and clean archticture.

## Features ✨

- ✈️ **Multi-Modules**
- 🥇 **Clean Architecture** 
- 🎨 **Compose UI** 
- 🚀 **Kotlin** 
- 🗄️ **Coroutines & Flow**
- 🚀 **Retrofit**
- 🌙 **Dark/Light Theme**
